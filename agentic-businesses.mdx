## Example agentic businesses

### Daily Sports Picks

**Customer Journey:**  
- User lands on your Whop store and sees a headline about today‚Äôs games  
- They subscribe to get daily picks posted in a private forum or chat  
- The AI posts daily betting advice, updates game commentary, and sends recaps  
- Users ask questions or engage ‚Äî AI replies or creates reactions  
- You retain users with content rewards, chat banter, and occasional DMs

---

**Step 1: Create a Whop**

Use the `create_whop` mutation to generate a store with a relevant title:

```graphql
mutation {
  createWhop(input: {
    name: "BetBot Daily Picks",
    description: "AI-generated daily sports betting picks and commentary"
  }) {
    id
    slug
  }
}
```

---

**Step 2: Create a Pricing Option**

```json
POST /api/users/v6/create-pricing-option
```

```json
{
  "whop_id": "WHOP_ID",
  "name": "Daily Picks Subscription",
  "price": 2000,
  "interval": "day"
}
```

---
