## Prerequisites

Before you start, make sure you have the following:

- **Discord Bot Token**
- **Whop API Key**
- **A place to host the bot.**

### Where to host your bot

There are many hosting options available for your Discord bot. Below are some recommendations:

- [Hop.io](https://hop.io) (Free Tier)
- [Pebblehost](https://pebblehost.com/bot-hosting) (Paid)
- [DigitalOcean](https://digitalocean.com) (Paid)
- [Heroku](https://heroku.com) (Paid)
- [Replit](https://replit.com) (Free Tier)
