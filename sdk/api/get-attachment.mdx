## Get Attachment Details

Use the following snippet to fetch details for an attachment using its ID.

```typescript
import { whopApi } from "@/lib/whop-api";

const attachmentDetails = (await whopApi.getAttachment({id: "id_xxx"})).attachment;

if (attachmentDetails) {
  console.log(attachmentDetails);
}
```
