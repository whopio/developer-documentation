## Transfer Funds

Use the following snippet to transfer funds. You'll need to provide details like source and destination ledger accounts, amount, and currency in the `TransferFundsInput`.

```typescript
import { whopApi } from "@/lib/whop-api";

// The TransferFundsInput structure needs to be defined based on your GraphQL schema.
// Example structure:
// interface TransferFundsInput {
//   sourceLedgerAccountId: string;
//   destinationLedgerAccountId: string;
//   amount: string; // Or number, depending on schema (often string for precision)
//   currency: string;
//   description?: string;
//   // ... other fields as required by your schema
// }

const initiateTransfer = async () => {
  const input = {
    sourceLedgerAccountId: "ledger_id_source",      // Replace with actual source ledger ID
    destinationLedgerAccountId: "ledger_id_dest",  // Replace with actual destination ledger ID
    amount: "100.00", // Example amount as string
    currency: "USD",
    description: "Payment for services rendered"
  };

  try {
    // The transferFunds mutation often returns a boolean or a transaction status/ID.
    // The provided GraphQL returns a boolean.
    const success = (await whopApi.transferFunds({input})).transferFunds;

    if (success) {
      console.log("Funds transfer initiated successfully.");
    } else {
      console.log("Funds transfer failed or is pending.");
    }
  } catch (error) {
    console.error("Error transferring funds:", error);
  }
};

initiateTransfer();
```
