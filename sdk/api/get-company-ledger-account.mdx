## Get Company Ledger Account

Use the following snippet to fetch the ledger account for a company using its ID.

```typescript
import { whopApi } from "@/lib/whop-api";

const ledgerAccount = (await whopApi.getCompanyLedgerAccount({companyId: "biz_xxx"})).company?.ledgerAccount;

if (ledgerAccount) {
  console.log(ledgerAccount);
}
```
