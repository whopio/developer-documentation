## Get Current User Details

Use the following snippet to fetch details for the current user.

```typescript
import { whopApi } from "@/lib/whop-api";

const currentUser = (await whopApi.getCurrentUser()).viewer?.user;

if (currentUser) {
  console.log(currentUser);
  // Example: Accessing balance cache
  // currentUser.ledgerAccount?.balanceCaches?.nodes.forEach(balance => {

}
```
