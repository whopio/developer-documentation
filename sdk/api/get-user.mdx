---
title: Get user
description: Retrieve the public profile information for any user on Whop
---

## Fetch a user via ID

Use the following snippet to fetch a user's public profile information by their ID. Their ID looks like `user_xxx`.


```typescript
import { whopApi } from "@/lib/whop-api";

const user = (await whopApi.getUser({userId: "user_xxx"})).publicUser;

console.log(user);
```




## Fetch a user via username

Use the following snippet to fetch a user's public profile information by their username.


```typescript
import { whopApi } from "@/lib/whop-api";

const user = (await whopApi.getUser({username: "jack"})).publicUser;

console.log(user);
```
