---
title: Get user
description: Retrieve the public profile information for any user on Whop
---

## Fetch a user via ID

Use the following snippet to fetch a user's public profile information by their ID. Their id looks like `user_xxxx`.


```typescript
import { whopApi } from "@/lib/whop-api";

const user = (await whopApi.getUser({userId: "user_xxxx"})).publicUser;

console.log(user);
```

```curl
curl --location 'https://api.whop.com/public-graphql' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_API_KEY' \
--data '{
    "operationName": "getUser",
    "variables": {
        "userId": "user_123"
    }
}'
```


## Fetch a user via username

Use the following snippet to fetch a user's public profile information by their username.


```typescript
import { whopApi } from "@/lib/whop-api";

const user = (await whopApi.getUser({username: "jack"})).publicUser;

console.log(user);
```
