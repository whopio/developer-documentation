---
title: Retrieve User
description: Retrieve the public profile information for any user on Whop
---

## Fetch a user via ID

Use the following snippet to fetch a user's public profile information by their ID. Their id looks like `user_xxxx`.


<CodeGroup>
```javascript Typescript
import { whopApi } from "@/lib/whop-api";

const user = (await whopApi.retrieveUser({userId: "user_xxxx"})).publicUser;

console.log(user);
```

```curl
curl --location 'https://api.whop.com/public-graphql' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_API_KEY' \
--data '{
    "operationName": "retrieveUser",
    "variables": {
        "userId": "user_123"
    }
}'
```

</CodeGroup>
