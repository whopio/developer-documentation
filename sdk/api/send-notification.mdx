## Send Notification

Use the following snippet to send a notification. You will need to construct the `SendNotificationInput` with details such as recipient(s), title, message, and any relevant links or metadata.

```typescript
import { whopApi } from "@/lib/whop-api";

// The SendNotificationInput structure needs to be defined based on your GraphQL schema.
// Example structure:
// interface SendNotificationInput {
//   userId?: string;
//   companyId?: string; // To send to all users of a company, or specific topic
//   topic?: string; // e.g., company_all, experience_xxxx
//   title: string;
//   message: string;
//   url?: string;
//   // ... other fields as required by your schema
// }

const sendPushNotification = async () => {
  const input = {
    userId: "user_xxx", // Or companyId/topic
    title: "Important Update",
    message: "Your new content is now available!",
    url: "https://example.com/new-content"
    // Add other necessary fields for SendNotificationInput
  };

  try {
    // The sendNotification mutation often returns a boolean or a status.
    const success = (await whopApi.sendNotification({input})).sendNotification;

    if (success) {
      console.log("Notification sent successfully.");
    } else {
      console.log("Failed to send notification or it's being processed.");
    }
  } catch (error) {
    console.error("Error sending notification:", error);
  }
};

sendPushNotification();
```
